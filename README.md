# 部署

## 步骤

1. 登录Okteto云，点击Deploy
2. 复制本项目的git地址 `https://github.com/vhqyeo50893/v2okteto.git`
3. 点击Git URL，粘贴git地址，点击`Deploy`

### 变量

对部署时需设定的变量名称做如下说明。

`uuid`：`ad806487-2d26-4636-98b6-ab85cc8521f7`

`端口`：`443`

`额外id`：`64`

`传输协议`：`ws`

`伪装域名`：`okteto app的域名或cf workers域名`

`路径`：`/`

`底层传输安全`：`tls`
